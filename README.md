This repository contains a code for smart locker
Problem- Sometimes unknown also get the key or password from somewhere and the thefts happens

 Solution 
.The smart locker-This smart locker ask permission from the owner to give permission for opening if the user allow the locker will get open with clicking picture who is try to open it 
and if the owner not allowed it show invalid and click the picture of it also
.It also help to know how many times locker will open and how many times try to.open but not get open
.It wil reduce theft
